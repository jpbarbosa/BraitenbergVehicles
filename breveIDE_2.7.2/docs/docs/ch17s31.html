Link

Description for class Link:

Link objects are physically simulated subclasses of Mobile. Link objects may be used in conjunction with Joint objects, to become pieces of an articulated body.

Links may also be associated with MultiBody objects, but this is not required (as they were prior to breve 2.0).

In breve 2.0 and later, Link is a subclass of Mobile, and MultiBody objects can be constructed out of any Mobile objects. This means that a great deal of the methods and functionality that used to be a part of Link are now accessed via the superclasses Mobile and Real.

Inherits from Mobile
