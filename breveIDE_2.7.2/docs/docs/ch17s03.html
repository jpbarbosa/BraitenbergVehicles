BraitenbergControl

Description for class BraitenbergControl:

This class is used for building simple Braitenberg vehicle simulations. To create a Braitenberg vehicle simulation, subclass BraitenbergControl and use the init method to create BraitenbergLight and BraitenbergVehicle objects.
