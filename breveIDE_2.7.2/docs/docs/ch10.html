Chapter 10. Saving Movies and Snapshots

breve allows you to save snapshots of your simulation as PNG files (the section called “Saving Snapshots”), and movies as MPEG files (the section called “Saving Movies”).

Not Available When Using breve_cli

These features are not supported for the "graphicless" build of breve, breve_cli.

Saving Movies

You can save MPEG movies of breve simulations using the class Movie.tz. First create an instance of the class, then begin recording a movie by using the method record. For your convenience, the standard Control class simulation menu includes an option for exporting an MPEG movie.
