BraitenbergLight

Description for class BraitenbergLight:

A BraitenbergLight is used in conjunction with BraitenbergControl and BraitenbergVehicle. It is what the BraitenbergSensor objects on the BraitenbergVehicle detect.

There are no special behaviors associated with the lights--they're basically just plain Mobile objects.
