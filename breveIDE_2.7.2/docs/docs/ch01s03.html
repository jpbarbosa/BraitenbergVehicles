Getting Help With breve

There are several ways to get help using breve:

  • Read the class documentation for more detailed information about each class. These class files are listed in the breve Class APIs.

  • Visit the breve discussion forums at http://www.spiderland.org/forums/.

  • Mail the author at jk [at] spiderland.org
